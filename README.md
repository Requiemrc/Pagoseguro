# PagoSeguro – Monorepo

Este repositorio contiene:

- **api/** → Backend (Node + Express)
- **admin/** → Panel administrativo (React + Vite)
- **app/** → Aplicación móvil (Expo)

## Cómo iniciar el proyecto

### 1. Backend
````bash
cd api
npm install
npm run dev


### 1. admin web
cd admin
npm install
npm run dev


### 1. app mobile
cd app
npm install
npx expo start
